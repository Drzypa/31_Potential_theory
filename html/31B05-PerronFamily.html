Perron family


Definition.

Let G⊂ℂ be a region, ∂∞⁡G the extended boundary of G and S⁢(G) the set of subharmonic functions on G, then if f:∂∞⁡G→ℝ is a continuous function then the set

𝒫⁢(f,G):={φ:φ∈S⁢(G)⁢ and ⁢lim supz→a⁡φ⁢(z)≤f⁢(a)⁢ for all a∈∂∞⁡G},

is called the Perron family.

One thing to note is the 𝒫⁢(f,G) is never empty. This is because f is continuous on ∂∞⁡G it attains a maximum, say |f|<M, then the function φ⁢(z):=-M is in 𝒫⁢(f,G).

Definition.

Let G⊂ℂ be a region and f:∂∞⁡G→ℝ be a continuous function then the function u:G→ℝ

u⁢(z):=sup⁡{ϕ:ϕ∈𝒫⁢(f,G)},

is called the Perron function associated with f.

Here is the reason for all these definitions.

Theorem.

Let G⊂C be a region and suppose f:∂∞⁡G→R is a continuous function. If u:G→R is the Perron function associated with f, then u is a harmonic function.

Compare this with Rado’s theorem (http://planetmath.org/RadosTheorem) which works with harmonic functions with range in ℝ2, but also gives a much stronger statement.
