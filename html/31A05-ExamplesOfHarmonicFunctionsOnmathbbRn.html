examples of harmonic functions on ℝn


Some real functions in ℝn (e.g. any linear function, or any affine function) are obviously harmonic functions. What are some more interesting harmonic functions?

  • •

    For n≥3, define (on the punctured space U=ℝn∖{0}) the function f⁢(x)=∥x∥2-n. Then

    ∂⁡f∂⁡xi=(2-n)⁢xi∥x∥n,

    and

    ∂2⁡f∂⁡xi2=n⁢(n-2)⁢xi2∥x∥n+2-(n-2)⁢1∥x∥n

    Summing over i=1,…,n shows Δ⁢f≡0.

  • •

    For n=2, define (on the punctured plane U=ℝ2∖{0}) the function f⁢(x,y)=log⁡(x2+y2). Derivation and summing yield Δ⁢f≡0.

  • •

    For n=1, the condition (Δ⁢f)⁢(x)=f′′⁢(x)≡0 forces f to be an affine function on every segment; there are no “interesting” harmonic functions in one dimension.
